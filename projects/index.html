Projects

Here's a small selection of some of my recent projects and experiences.

Huddle Landing Page

huddle landing page is a mockup website from FrontendMentor.io, it was built using simple HTML and css the biggest challenge was lining up the SVG so that it works on all screen sizes.

Shadow Foundry.ai

Shadow foundry is a website i made for a client, it's a static HTML/SCSS/JS website with the splide carousel library and self hosted fonts.

Rock Paper Scissors

Rock Paper Scissors is a FrontendMentor.io challenge where you play rock paper scissors against the Computer which is simply a 1/3 chance of picking any, it was built using React, typescript and CSS3

TODO Page

TODO page is also a FrontendMentor.io challenge, it was built using React and CSS3, You can add/remove tasks or mark as done there is a dark and light theme with different backgrounds.

Linus tech tips

LTT is a university project to build a shopping page with a cart and details page and some filtering functionality, The name Linus Tech Tips is from a Canadian tech Youtuber, this website was built using PHP and an MYSQL database, and plain CSS3.
